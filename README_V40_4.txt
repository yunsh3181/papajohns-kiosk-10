PAPA OS v40.4 관리자 주문 표시 수정

원인
- 키오스크 주문 상태는 payment_pending(결제대기)으로 저장됨
- 관리자 기본 '진행 중' 필터는 new/accepted/cooking만 표시해 주문이 숨겨졌음
- 관리자 버튼도 이전 신규/접수 프로세스를 사용하고 있었음

수정
- 진행 중 필터에 결제대기/결제완료/조리중/조리완료 포함
- 결제대기 → 결제완료 → 조리시작 → 조리완료 → 수령완료 버튼 적용
- Firestore createdAt이 없는 과거 주문도 표시되도록 쿼리 개선
- 브라우저에서 최신순 정렬
- 고객번호를 휴대전화 뒤 4자리로 저장
