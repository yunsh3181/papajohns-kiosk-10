PAPA JOHNS v36 데모

신규 기능
- 사용중 테이블에 줄서기 버튼 추가
- 줄서기 인원/전화번호 입력
- Firestore waitlist 컬렉션에 대기 등록
- 관리자에 줄서기 탭 추가
- 대기 고객 호출 / 입장 / 취소
- 신규 줄서기 등록 시 관리자 알림음
- 음료 카드에 ZERO / ORIGINAL 배지
- 음료 카드에 500mL / 1.5L 용량 배지
- 코카콜라/스프라이트 계열 색상 구분
- 스프라이트 제로 500mL, 1.5L 추가
- 제공된 스프라이트 제로 이미지를 1.5L에 적용
- 500mL는 임시로 동일 이미지 사용 (정확한 500mL 사진 수신 후 교체)
