수정사항
1. 로고와 배경 파일명/경로 수정
2. 모바일 하단 '처음으로' 고정 버튼 추가
3. 세트메뉴 3개 카드 한 화면 배치
4. 세트 크러스트 3개 카드 한 화면 배치
5. 사이드 카드 축소
6. 음료 3x2 배열
7. 포장 바로주문/예약/해피아워 한 화면 배치
8. 예약은 현재 시각 30분 이후부터 가능
9. 예약 시간선택 화면 축소
10. 전화번호 입력화면 축소
11. 주문 관리자와 좌석관리 한 페이지 통합

13. 세트 필수 사이드에서 콘샐러드/코울슬로 제외
14. 관리자 알림음 확대
