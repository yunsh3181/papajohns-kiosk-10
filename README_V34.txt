PAPA JOHNS v34 좌석 로직 변경

Bottle Zone
- 4인석은 최대 5명까지 선택 가능
- 2인석은 기존대로 최대 2명

Room Zone
- 최초 이용은 6~12명만 가능
- 총 3테이블 / 테이블당 4명
- 6~8명 입실 시 2테이블 사용
- 9~12명 입실 시 3테이블 사용
- 기존 팀이 있으면 남은 테이블 수 × 4명까지 추가 가능
- 예: 6명 이용중 → 1테이블 남음 → 최대 4명 추가 가능

Outside Zone
- 총 4테이블 / 테이블당 4명 / 최대 16명
- 사용중 테이블 1개당 추가 가능 인원 4명 감소
- 예: 1테이블 사용중 → 최대 12명 추가 가능
- 5명 이상 선택 시 필요한 빈 테이블을 자동으로 묶어서 배정
