# Papa OS CHANGELOG

## v40.11 — 2026-07-14 21:30 KST

### 별관 좌석 선택 화면
- 야외석과 파파존 화면은 변경하지 않음
- 별관 전용 실제 배치도형 좌석 선택 화면 추가
- 좌석 위치 확정
  - 1번: 창가석 1 · 최대 2인
  - 2번: 별관 2 · 세로형 테이블 · 최대 5인
  - 3번: 별관 3 · 사선형 단체석 · 최대 5인
  - 4번: 커플석 · 장식장 바깥편 · 최대 2인
- 좌석 상태별 색상 표시
  - 초록: 이용 가능
  - 노랑: 선택 중
  - 빨강: 사용 중
- 인원 초과 좌석 자동 비활성화
- Firestore 기존 좌석 상태 및 선택 기능 유지

## v40.10 — 2026-07-14

## v40.12 (2026-07-14 22:04 KST)
- 전화번호 입력 화면 하단의 중복 `선택 완료` 푸터를 숨김 처리했습니다.
- 주문 유형 선택을 포함한 모든 주문 섹션에 어두운 그린·블랙 테마를 적용했습니다.
- 2·3·4인 세트 카드의 메뉴 사진을 제거하고 간단한 이모지 구성으로 변경했습니다.
- 2인 세트: 피자 + 파스타 + 작은 음료
- 3인 세트: 피자 + 사이드 1종 + 큰 음료
- 4인 세트: 피자 + 사이드 2종 + 큰 음료


## v40.14 (2026-07-14 22:45 KST)

### 테스트 편의 및 UI 가독성 개선
- 전화번호 `선택하기` 버튼을 숫자패드 4행의 `0` 오른쪽에 고정
- 전화번호 선택 버튼을 활성/비활성 상태 모두 파파존스 레드 계열로 표시
- 피자 구성의 `한 판` 및 `하프앤하프` 제목·설명 글자 대비 강화
- 오리지널·골드링·치즈롤·씬 도우/크러스트 글자 색상을 흰색과 밝은 회색으로 강제
- 하프앤하프 미리보기 피자 이미지의 흰 배경을 주변 다크 톤에 혼합
- 테스트 중 10초 경고 및 20초 좌석 자동 해제 기능 비활성화 (`DISABLE_IDLE_RESET=true`)
- 일반 주문 도우 타입 선택 카드를 데스크톱·키오스크에서 크게 확장

## v40.15 — 2026-07-14 23:40 KST
- 전화번호 11자리 입력 완료 시에만 `선택하기` 버튼 활성화
- 주문 유형 및 2·3·4인 세트 카드의 제목·설명 글자 대비 강화
- 먹고가기 주문에서 `이전` 버튼을 누르면 좌석을 해제하지 않고 바로 전 단계로 이동
- 일반 주문의 도우·사이즈·크러스트 선택을 한 화면으로 통합
- 선택한 도우와 사이즈에 따라 이용 가능한 크러스트 자동 활성/비활성화
- 피자·사이드·음료 이미지의 흰 배경이 다크 카드에서 두드러지지 않도록 배경 블렌드 처리
- 음료 용량별 수량을 `− / 수량 / ＋` 방식으로 직접 조절하도록 변경
- 주문 확인 화면에 일반 주문의 사이드 및 음료 금액 표시 추가
- 관리자 결제대기 주문 알림을 기본 활성화하고 첫 화면 터치 시 오디오 잠금 자동 해제
- 관리자 새 주문 발생 시 알림음 재생 실패 여부와 무관하게 토스트 및 탭 제목 알림 표시
- 브라우저 캐시 갱신을 위해 index/admin 리소스 버전을 v40.15로 변경
