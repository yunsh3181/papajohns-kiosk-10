# Papa John's Kiosk Changelog

## v41.2 (2026-07-15)

### 선택 UI 통일
- 피자 구성, 도우, 사이즈, 크러스트 선택 시 노란색 외곽선과 `✓ 선택됨` 표시
- 토핑, 사이드, 음료에서 수량이 1개 이상이면 동일한 선택 상태 표시
- 일반 주문과 세트 추가 음료 모두 4개 음료 카드 구조로 통일
- 음료 카드 안에서 큰 용량과 작은 용량의 수량을 조절하도록 유지
- 음료 이미지 크기와 중앙 정렬 유지
- 하프앤하프 선택 시 레귤러 사이즈 선택 불가 규칙 유지

### 좌석 관리자
- 전체 정리 버튼을 누르면 사용 중, 정리 중, 선택 중 좌석을 즉시 빈자리로 변경
- 예약 좌석은 전체 정리 대상에서 제외
- 빈자리 카드에서 사용 중 또는 예약 등록 가능
- 예약자명, 인원, 예약 일시, 전화번호 등록 및 수정 가능
- 예약 좌석을 사용 중으로 전환하거나 예약 취소 가능
- 예약 좌석은 파란색으로 표시
- 고객 화면에서는 예약 30분 전부터 해당 좌석을 예약 상태로 비활성화

### 초기화면 이벤트 바로가기
- 해피아워 카드를 누르면 해피아워 주문으로 바로 연결
- 해피아워 운영시간 외에는 16:00~20:00 범위의 예약 주문 화면으로 연결
- 포장 할인 카드를 누르면 포장 주문 설정 후 포장 20% 할인으로 바로 연결
