# Papa John's Kiosk Changelog

## v40.10 (2026-07-14 12:09)

### 좌석 선택 화면
- 야외석을 실제 배치 순서로 변경: 파파존스 입구 → 1 → 2 → 3 → 4 → 보틀 입구
- 파파존스 입구와 보틀 입구는 선택 불가능한 위치 안내 표식으로 분리
- 야외 테이블 1~4번 카드형 UI 및 실제 야외 사진 적용
- 파파존 좌석을 바테이블석 │ 출입구 │ 커플석 순서로 변경
- 바테이블석 최대 4인, 커플석 1~2인 추천 표기
- 커플석과 바테이블석 실제 사진 적용
- 기존 별관·별관룸 선택 기능 유지
- 먹고가기 테스트 모드 시간 제한 해제 유지
