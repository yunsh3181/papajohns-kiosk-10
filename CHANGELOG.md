# Papa John's Kiosk Changelog

## v41.3 — 2026-07-15

### 모바일 전용 최적화
- PC용 `style.css`, `admin.css`, `seats.css`는 수정하지 않음.
- 모바일 키오스크 전용 `mobile.css` 추가.
- 좌석 관리자 전용 `seats-mobile.css` 추가.
- 주문 관리자 전용 `admin-mobile.css` 추가.
- 모바일에서 초기화면, 진행 인디케이터, 메뉴 카드, 도우·크러스트, 음료 4카드, 전화번호 키패드, 주문 확인, 하단 버튼 영역의 잘림과 가로 넘침을 보정.
- 모바일 좌석 관리자와 주문 관리자를 한 열 중심의 큰 터치 UI로 보정.
- 390px 이하 소형 휴대폰에 추가 레이아웃 적용.
