# Papa John's Kiosk Changelog

## v41.5 — 2026-07-15
### 모바일 주문 편의성 중심 개편
- PC용 `style.css`, `admin.css`, `seats.css`는 수정하지 않음.
- 모바일 전용 `mobile.css`만 개편.
- 모바일 피자·사이드·음료 이미지를 작은 썸네일 중심으로 축소.
- 도우·크러스트 카드를 세로형 대형 이미지 대신 가로형 텍스트 중심 카드로 변경.
- 일반주문 및 세트 추가 음료의 4개 음료 카드 구조와 주문 로직은 그대로 유지.
- 음료 이미지를 동일한 크기로 통일하고 용량 선택 버튼을 우선 노출.
- 사이드 메뉴 카드를 한 줄형으로 압축하고 수량 `− / 숫자 / ＋` 버튼은 크게 유지.
- 세트·프로모션·주문유형 카드는 이미지 대신 아이콘과 핵심 문구 중심으로 축소.
- 하프앤하프 조합 이미지는 식별 가능한 최소 크기로 유지.
- 좌석 구역 이미지는 방향 확인용 썸네일 크기로 축소.
- 주문, 가격 계산, 좌석, 장바구니, 관리자 연동 JavaScript 로직은 변경하지 않음.

## v41.3 — 2026-07-15
### 모바일 전용 최적화
- PC용 `style.css`, `admin.css`, `seats.css`는 수정하지 않음.
- 모바일 키오스크 전용 `mobile.css` 추가.
- 좌석 관리자 전용 `seats-mobile.css` 추가.
- 주문 관리자 전용 `admin-mobile.css` 추가.
- 모바일에서 초기화면, 진행 인디케이터, 메뉴 카드, 도우·크러스트, 음료 4카드, 전화번호 키패드, 주문 확인, 하단 버튼 영역의 잘림과 가로 넘침을 보정.
- 모바일 좌석 관리자와 주문 관리자를 한 열 중심의 큰 터치 UI로 보정.
- 390px 이하 소형 휴대폰에 추가 레이아웃 적용.
